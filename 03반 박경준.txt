w/a/s/d: 상하좌우 움직이기

<메뉴명령어>

easy:쉬움 난이도

invinciblity:무적모드(플레이어가 죽지않음)

<구현한 내용>
카메라효과/이펙트/콤보/점수출력/총알(잔상)/ 
적(랜덤한 위치에서 생성 ,x,y중 플레이어와 가까운 쪽으로 접근, 타일 검은색 변경,

게임오버시 화면전환/
총알,적,주인공 사망이펙트/

장애물 통과 가능 적, 못통과하는 적(색으로 구분)/

총알 팩(총알을 채워 주기만 하는 팩 일 수도 있고 총알을 채워주면서 특수 총알도 채워주는 팩 일 수 도 있음 )

적 구현 내용: 난이도에 따른 보드 크기에 맞추어 생성, 보드의 랜덤위치에서 생성, 적이 총알과 부딪혔을시 사망(총알의 위치 값과 적의 위치값비교), 장애물 통과 가능과 불가능은 move시 장애물과 충돌하는 적이면 통과 아니면 위치를 롤백해서 구현함, 이동은 플레이어의 중심점과 적 캐릭터의 중심점 두개의 점을 적 캐릭터 기준으로 선형변환해서 상대적 위치를 구하고 상대 위치의 부호를 계산해서 방향을 구하고 그방향으로 적을 조금씩 이동 시킨다.

플레이어 구현 내용: 보드의 크기에 맞추어 사이즈 조정, 몸에 circle 6개를 플레이어 중심으로 돌아가게 구현, 플레이어 이동은 게임 보드밖으로 못나가게 충돌체크, 자신의 몸 위치의 보드의 색이 흰색이 아닐경우 이동을 못하게 함, 플레이어의 움직임을 부드럽게 하기위해 타이머에 딜레이를 줘서 움직임+상하좌우 동시입력받게 구현

보드 구현 내용: 젤위의 스코어 보드를 기준 그 아래의 정사각형의 보드를 그림, 모든 보드의 칸은 원래 검은색으로 설정, 특정 조건들을 만족하면 새이 변하게 구현함, 화면의 흔들리는 효과는 보드만이 흔들리므로 보드를 흔들어서 구현(흔들리는 효과 발생시 상하 좌우로 보드를 이동하는데 고속으로 프린트)

총알 구현 내용: 총알이 바로 쭉 나가는 것이 아니라 총알 마다 스피드와 딜레이 이동거리에 차이를 두어서 시간이 지날수록 빨라지는 일종의 레이져 같은 느낌으로 구현함, 기본 총알은 검은색 그라데이션으로 특수 총알은  빨간색 그라데이션으로 표현함 적이 맞거나 보드의 끝에 도달하면 총알 삭제, 총알은 여러 군데에서 쓰이고 한번에 여러개가 존재할 것이기 때문에 총알 관리 클래스를 만들어서 총알의 이동과 프린트를 함, 각 총알이 발사 될때 플레이어의 특수 총알 갯수를 받아와서 특수총알 발사를 체크한다. 특수총알은 2개에서 3개의 총알이 날아가면서 장애물에 부딪힐때 장애물을 파괴하면서 지나간다.

이펙트: 적과 플레이어가 생성 될때 나오는 이펙트이며 간단히 생성 위치 점 한개와 그주변에 원을 그리고 시간이 지날때마다 크기를 점차 줄이면서 이펙트 구현, 적의 이펙트는 이동과 충돌에 플레이어의 이펙트는 이동에 관련되어있어 타이머에서 프레임을 줄이면 그만큼 이펙트의 속도도 줄어든다.

생명력: 기본 플레이어 생명력은 3이고 그것이 스코어 보드에 출력 된다.
스코어 보드: 외쪽 위에 스코어가 올라가고 있고 적을 죽이면 점수가 추가되며 콤보 발생시 콤보!!라고 출력한다. 가운데에는 플레이어의 생명력을 출력한다. 오른쪽에는 현재 가지고 있는 특수 총알개수를 표현한다.




메뉴 설명: START 누르기전에 난이도 선택 가능, 게임오버시 난이도 선택후 START 누르면 다시 시작
무적모드: 말그대로 무적모드여서 적들과 부딪혀도 죽지 않음
일시정지: 한번누르면 일시정지, 다시 누르면 일시정지 해제
게임종료: 게임종료
